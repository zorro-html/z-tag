# `<z-tag>`

Tags & badges element

## Attributes

- `primary`: with primary style
- `badge`: more border-radius than default, just like App badges in iOS

## Examples

```
<p>
  tags:
  <z-tag primary>html</z-tag>
  <z-tag>css</z-tag>
  <z-tag>javascript</z-tag>
  <z-tag></z-tag>
  badges:
  <z-tag badge primary>100</z-tag>
  <z-tag badge>10</z-tag>
</p>
```
